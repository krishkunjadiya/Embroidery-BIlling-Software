# **App Name**: Embroidery Billing Software

## Core Features:

- Bill Input Form: Capture bill details: Implement a form to input bill components like item descriptions, quantities, rates, taxes (GST), discounts, and total amount in Indian Rupees. Mimic the layout from the example bill.
- Real-time Preview: Real-time Preview: Show a real-time preview of the bill as the user fills out the form. Display all the components and the total amount in Indian Rupees, updating dynamically.
- Print Button: Print Functionality: Integrate a print button that allows users to print the generated bill in a clean and formatted layout.

## Style Guidelines:

- Primary color: White or light gray for the background to ensure readability.
- Secondary color: Light blue or green (#E0F7FA or #E8F5E9) for form fields and interactive elements.
- Accent: Teal (#008080) for buttons and important actions.
- Use a clear and readable font for all text elements.
- Maintain a clean and organized layout, mirroring the structure of a standard Indian bill.
- Use simple, recognizable icons for common actions like 'Print' and 'Clear'.
